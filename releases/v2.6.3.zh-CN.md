# Prompt Optimizer v2.6.3

## 概括
- 新增图像提示词提取能力，可以把已有图片重新转成可复用的提示词素材。
- Core 层补上了更完整的图像理解服务与提取模板支撑。
- 桌面端发布链路更稳了，同时也更新了默认 MiniMax 模型基线。

## 亮点
- 图像提示词提取正式成为一套工作流，更适合从生成图或参考图中反向整理提示词。
- 核心图像理解能力进一步补强，为提取流程提供了更完整的底层支持。
- 通过修复桌面端发布流程和更新 MiniMax 默认模型，整体发布稳定性更进一步。

## 产品更新
### Web
- 在 UI 中新增了图像提示词提取工作流。
- 让图像提取行为可以直接复用新的图像理解能力和提取模板。

### Core/Infra
- 在 Core 中新增图像理解服务以及面向提取场景的模板。
- 将默认 MiniMax 模型基线升级到 `M2.7`。
- 清理了发布自动化逻辑，避免桌面端构建任务意外自动发布 Release。

## 修复
- 修复了桌面端发布流程，避免构建任务自动发布 Release。
- 通过更新 MiniMax 默认模型，降低默认模型配置过旧带来的行为偏差。

## 破坏性变更 / 升级说明
- 无。这一版扩展的是图像能力，不需要额外迁移步骤。

## 开发者说明
- 如果你想追溯当前代码库里“图像提示词提取”正式落地的起点，这一版就是关键版本。
- 这一版也标志着图像理解能力从相对独立的逻辑，开始转向可复用、可发布的正式工作流。

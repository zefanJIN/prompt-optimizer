# Prompt Optimizer v2.11.0

## 概括
- 新增面向桌面端与浏览器运行时的部署感知远程备份能力。
- 强化资源完整备份的恢复安全性，覆盖收藏与图片素材。
- 扩展图像工作流，加入 OpenAI 多图编辑支持和 Prompt Garden 图生图指南。
- 改进外部链接与复制后打开操作，让浏览器行为更安全、失败反馈更清晰。

## 亮点
- 远程备份现在成为更完整的数据安全工作流：用户可以配置 provider 存储，备份带图片资源的应用数据与收藏，并在恢复前经过更严格的资源校验。

## 产品更新
### Desktop
- 新增桌面主进程远程存储能力，支持 S3 兼容服务与 WebDAV provider。
- 通过 preload bridge 暴露远程存储 IPC，让桌面端恢复流程避开仅依赖 renderer 网络请求的路径。
- 补充远程备份配置示例，并增加 provider 路由相关的桌面 IPC 覆盖。

### Web
- 在数据管理器中加入部署感知的远程备份界面，覆盖已保存设置、自动连接、恢复控制与存储提醒。
- 新增应用数据、收藏、图片资源和远程快照的资源完整备份与恢复辅助能力。
- 为 AI 平台输出加入复制后打开操作，并统一使用共享的外部链接处理逻辑。
- 新增 Prompt Garden 图生图工作流指南入口。

### Extension
- 扩展包版本同步到 v2.11.0，并继承共享 UI 改进。

### Core/Infra
- 新增 OpenAI 多图编辑请求支持，并补充 adapter 覆盖。
- 新增共享的图片资源恢复校验、哈希与 package 辅助能力。
- 补充远程备份、快照恢复、资源包、外部链接和桌面 handler 的单测与集成覆盖。

## 修复
- 远程备份恢复会在导入应用数据前防护缺失或损坏的图片字节。
- 数据管理器会自动连接已保存的远程备份设置。
- 外部链接会经过协议校验，并使用更安全的 `noopener,noreferrer` 打开方式。
- 复制后打开失败时会展示反馈，不再静默失败。
- 历史链元数据在可能过期时会被清理。

## 破坏性变更 / 升级说明
- 无。

# Prompt Optimizer v2.9.1

## 概括
- 多图上传卡片的删除入口重新回到图片右上角，不再停留在底部的“删除”文字按钮上。
- 删除操作改成更轻量也更统一的 `X` 图标按钮，视觉上和旁边的拖拽句柄更协调。
- 按钮的对比度、点击区域和悬浮反馈都做了加强，在多图编辑时更不容易看漏。

## 亮点
- 多图卡片现在提供更醒目的右上角删除按钮，整理、替换多张参考图时会更顺手，也更符合直觉。

## 产品更新
### Desktop
- 桌面端会随打包后的 Web 工作区一起继承这次多图上传热修复，因此桌面应用中的交互表现也会同步更新。
### Web
- 多图上传卡片里的底部“删除”文字动作被替换为覆盖在预览右上角的图标按钮。
- 右上角覆盖按钮的样式做了增强，在不打断拖拽排序体验的前提下更容易被发现。

## 修复
- 修复了 `v2.9.0` 中多图删除操作虽然更明确、但底部文字按钮显得突兀且在密集编辑里不够顺手的问题。
- 修复了卡片内部操作层级不够统一的问题，让删除和排序入口重新回到同一套交互语义里。

## 破坏性变更 / 升级说明
- 无。

## 开发者说明
- 删除按钮重新作为预览覆盖层渲染，同时补充了源码守卫测试，约束它继续留在预览区而不是再次漂回底部 footer 行。

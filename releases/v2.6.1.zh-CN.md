# Prompt Optimizer v2.6.1

## 概括
- 模型管理能力进一步增强，支持克隆配置、恢复操作标签，并加强了自定义模型配置能力。
- 图像生成界面开始展示更丰富的运行时信息，包括 token 使用量和图片元数据。
- 修复了桌面端缺失的 IPC bridge，让提示消息链路更完整。

## 亮点
- 对复杂模型配置场景更友好了，尤其是在复制和微调多组模型配置时更顺手。
- 图像生成结果现在会直接展示更多有用信息，便于排查和比较。
- Core 对环境变量驱动配置和动态拉取模型的支持又向前推进了一步。

## 产品更新
### Desktop
- 补上了桌面端流程所需的 prompt message IPC bridge。

### Web
- 为图像生成增加了 token 使用量和图片元数据显示。
- 在 Model Manager 中新增了模型配置的 Clone 按钮。
- 恢复了模型管理器里缺失的操作标签，让关键动作更容易理解。

### Core/Infra
- 新增对环境变量加载自定义模型参数的支持。
- 为 Gemini 图像适配器启用了动态模型拉取能力。

## 修复
- 修复了桌面端提示消息链路缺失的 IPC 连接。
- 恢复了 UI 中丢失的模型管理器操作标签。

## 破坏性变更 / 升级说明
- 无。现有模型配置默认可继续工作，同时你会获得更灵活的配置方式。

## 开发者说明
- 这一版延续了“模型配置更动态化”的方向，尤其适合通过环境变量管理 provider 的团队。
- 如果你在追溯图像生成元数据首次进入应用 UI 的时间点，这一版就是关键版本。

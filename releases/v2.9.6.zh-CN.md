# Prompt Optimizer v2.9.6

## 概括
- 增强收藏功能：支持可复现性示例编辑、媒体管理，以及一键将示例应用到工作区会话。
- 用户现在可以在收藏编辑器中直接编辑可复现性变量、带参数和图片的示例。
- 收藏示例可以应用到 pro-variable 和图像工作区，自动预填模板变量和输入图。
- 将旧的 FavoriteCard 组件合并到编辑器表单中，统一创建和编辑体验。
- 强化收藏页面路由、工作区导航守卫和 Garden 收藏去重逻辑。

## 亮点
这一版将收藏从静态提示词升级为可复现模板：用户可以附加变量和包含参数、图片的示例，然后一键将示例应用到对应工作区。

## 产品更新
### Desktop
- 本次发布没有桌面端专属的用户可见变化。

### Web
- 收藏编辑器中新增可复现性示例编辑功能，支持媒体（输入图和输出图）。
- 收藏详情面板的每个可复现性示例卡片上增加"应用此示例"按钮。
- 点击"应用此示例"会将收藏加载到匹配的工作区，并自动将示例参数填入临时变量、将示例图片填入输入图。
- 支持的目标工作区：pro-variable、pro-multi-message、image-text2image、image-image2image、image-multiimage。
- 将旧 FavoriteCard 组件合并到 FavoriteEditorForm，统一创建和编辑交互。
- 新增从工作区到收藏页的按模式筛选路由及返回路径。

### Extension
- 本次发布没有扩展端专属的用户可见变化。

### Core/Infra
- 新增 `parseFavoriteReproducibility` 工具函数和 `FavoriteReproducibilityExample` 类型。
- 新增 `resolveAssetIdToDataUrl` 和 `normalizeImageSourceToPayload` 图片解析辅助函数。
- 扩展工作区 session API（proMultiMessageSession、proVariableSession、imageText2ImageSession、imageImage2ImageSession、imageMultiImageSession）以支持示例参数注入。
- 为可复现性展示组件、编辑器、示例应用流程补充完整的单元测试和 E2E 测试。

## 修复
- 修复同一快照在多个预览面板中重复渲染 Garden 收藏详情的问题。
- 修复收藏工作区列表项路由与导航守卫的兼容性问题。
- 修复 `useFavorite` 签名变更后收藏相关组件测试中缺少 `undefined` 第二参数断言的问题。

## 破坏性变更 / 升级说明
- 本次发布没有引入 API 或存储格式层面的破坏性变更。
- `useFavorite` / `use-favorite` 签名新增可选的第二个 `options` 参数，现有调用方无需修改。

## 开发者说明
- `FavoriteCard.vue` 已移除，请使用 `FavoriteEditorForm.vue` 配合 `FavoriteWorkspaceListItem` 进行收藏创建/编辑/列表交互。
- 示例应用管道与架构解耦：`applyFavoriteExample` 通过基于 key 的映射（`getFavoriteTargetKey`）定位目标工作区会话，便于扩展到新的工作区模式。

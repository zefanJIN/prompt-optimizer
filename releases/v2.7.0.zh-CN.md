# Prompt Optimizer v2.7.0

## 概括
- 新增了结构化对比评估工作流，让提示词对比更容易测试、校准和复用。
- 对比体验在测试面板、回退逻辑和工作区状态同步上都更顺了。
- 文档、官网展示和扩展商店素材也做了明显刷新，更方便理解和传播产品。

## 亮点
- 结构化对比评估正式成为一套完整工作流，可直接用于提示词测试与分析。
- 对比相关的交互体验更轻、更稳，结果面板和工作区状态协同也更自然。
- 项目的对外展示面焕然一新，包括独立营销站、更新后的演示内容，以及本地化扩展商店素材。

## 产品更新
### Web
- 在测试面板中加入了结构化对比工作流，并简化了整体对比评估体验。
- 让工作区测试状态在语言切换和恢复流程中保持响应式同步。
- 优化了 About / 资源区域，并在不具备参考上下文时隐藏容易误导的对比提示。

### Extension
- 补充了本地化商店截图，并刷新了面向商店展示的衍生 logo 资源。

### Core/Infra
- 新增结构化对比评估管线和校准产物，为更稳定的对比输出打下基础。
- 优化了评估重写行为，并让对比设计文档与新工作流保持一致。
- 在重做双语文档与 onboarding 体验的同时，也提升了文档构建稳定性。

## 修复
- 在缺少参考上下文时，隐藏了容易误导用户的 teacher 风格对比提示。
- 修复了对比版本回退逻辑，改为更稳定地使用会话记录 ID。
- 清理了影响文档部署流程的 MkDocs 构建问题。

## 破坏性变更 / 升级说明
- 无。现有数据与工作流默认不需要手动迁移。

## 开发者说明
- 这一版可以视为结构化对比架构的正式落点。如果你维护评估提示词或对比相关 UI，建议从这一版开始作为新基线。
- 这一段提交里也包含营销站和文档面的工作，但真正的产品主轴仍然是结构化对比能力。

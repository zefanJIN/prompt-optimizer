# Prompt Optimizer v2.10.1

## 概括
- 新增完整的 MkDocs 文档站，双语支持覆盖所有主要功能，包括 Prompt Garden、辅助功能、收藏、模型、测试和部署。
- 修复 RouterView 重构导致的 Pro 工作区操作失效（工具管理器、变量管理器、收藏、提示词预览），恢复 provide/inject 事件链。
- 修复英文文档中的中文残留，修正 Prompt Garden 导航路径。
- 更新项目落地页文案，简化发布流程。

## 亮点
本次补丁版本为 v2.10.0 功能带来了完整的文档站，并修复了 RouterView 重构引入的 Pro 工作区操作回归问题。

## 产品更新
### Web
- 在 `mkdocs/` 目录下新增 MkDocs 文档站，支持中英文双语。
- 新增 Prompt Garden 文档，覆盖导入码、使用示例、直接使用与收藏保存的工作流。
- 新增辅助功能文档，包括智能变量填充、文生图复刻和风格学习。
- 新增收藏文档，说明示例应用到工作区的方式和资源感知资产管理。
- 新增模型管理文档，包含 provider 请求详情说明和能力标签指引。
- 新增测试评估文档，覆盖"全部运行"并行执行说明。
- 新增桌面部署文档，说明 localhost/私有网段直连路由。
- 更新落地页文案，从"5大场景"调整为"4类工作区"。

### Core/Infra
- 新增 mkdocs-dev.yml 配置，包含文档站完整的导航结构。
- 新增 release notes 工作流文档和校验脚本。
- 更新项目状态、PRD 和 README 至 v2.10.0 功能集。

## 修复
- 修复 RouterView 重构导致的 Pro 工作区操作失效（openToolManager、openVariableManager、handleSaveFavorite、saveToGlobal、openPromptPreview），通过从 emit 迁移到 provide/inject 模式恢复。
- 修复 ContextUserWorkspace.vue 中 `@open-variable-manager` 事件未触发的问题，添加 provide/inject 处理并保留 emit 回退。
- 修复英文文档中残留的中文词汇（replicate.md、smart-fill.md、style-learn.md）。
- 修复 index 页面中 Prompt Garden 链接路径，从 `/basic/garden` 改为 `/basic/prompt-garden`。
- 修复 mkdocs-dev.yml 导航，将 optimization 拆分为系统提示词和用户提示词工作区页面。
- 移除 ContextUserWorkspace.vue 中未使用的 `open-tool-manager` emit 和 `appOpenPromptPreview` inject。
- 解决 P1/P2/P3 review 发现的问题：移除未使用的 inject、修复 saveToGlobal 绕过持久化、移除尾部空白。
- 简化发布文档中的 macOS 安全提示。

## 破坏性变更 / 升级说明
- 无。

## 开发者说明
- Pro 工作区操作现在使用 provide/inject 模式并带有 emit 回退，与其他工作区操作保持一致。`openToolManager` emit 已从 ContextUserWorkspace.vue 中移除。
- MkDocs 文档站可在 `mkdocs/` 目录下使用 `mkdocs serve` 本地构建。开发环境使用 `mkdocs-dev.yml` 配置。
- Release notes 必须覆盖从上一个 tag 到 HEAD 的完整范围，而不仅仅是最近的提交。使用 `git log v<上一个版本>..HEAD` 和 `git diff v<上一个版本>..HEAD --stat` 来验证覆盖范围。

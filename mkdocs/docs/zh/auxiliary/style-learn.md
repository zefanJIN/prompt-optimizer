# 风格学习

风格学习功能可以从参考图中提取风格、构图和色彩特征，同时保留你当前提示词的主体内容。

## 什么时候用

- 已有提示词主体，想学习参考图的特定风格
- 想保持提示词的核心内容，只改变视觉风格
- 需要从多个参考图中学习统一的风格特征

## 怎么用

### 步骤

1. 进入文生图工作区
2. 先输入你想画什么（提示词主体）
3. 上传参考图（点击"参考图"按钮）
4. 选择 **风格学习** 模式
5. 点击开始处理
6. AI 分析参考图风格并生成风格化提示词
7. 预览生成结果
8. 点击 **应用到当前提示词**

### 前置条件

- 必须先输入提示词主体内容
- 提示词不能为空
- 参考图清晰度要足够

### 界面说明

- **参考图区域**：显示上传的参考图缩略图
- **生成后的提示词**：保留主体 + 学习风格后的提示词
- **提取到的变量**：从图像中提取的风格相关变量
- **应用按钮**：将结果写入当前工作区

## 工作原理

### 分析过程

AI 会分析参考图的：

- 色彩搭配和色调
- 构图方式和视角
- 光影效果和氛围
- 艺术风格和技法

但不会改变：

- 提示词的主体内容
- 核心描述和关键元素

### 产出物

分析完成后会生成：

- **风格化提示词**：主体内容 + 学习到的风格特征
- **风格变量**：从图像中提取的风格参数（如色调、构图、风格）

### 变量处理

- 提取的变量专注于风格特征
- 变量名具有语义（如"色调"、"构图"、"风格"）
- 可以修改变量值来调整风格强度

## 使用场景

### 场景一：学习色彩风格

```
原始提示词：
一只橘猫坐在窗台上，阳光洒在身上

参考图：一张暖色调、柔和光影的摄影作品

生成的风格化提示词：
一只橘猫坐在窗台上，阳光洒在身上，暖色调，柔和的光影过渡，
温馨的氛围，摄影风格

提取的变量：
- 色调：暖色调
- 光影：柔和过渡
- 氛围：温馨
- 风格：摄影
```

### 场景二：学习构图方式

```
原始提示词：
一座雪山，山顶有积雪

参考图：一张采用低角度拍摄、强调前景的风景照

生成的风格化提示词：
一座雪山，山顶有积雪，低角度拍摄，强调前景，
广阔的视野，壮丽的氛围，风光摄影风格

提取的变量：
- 视角：低角度
- 构图：强调前景
- 氛围：壮丽
- 风格：风光摄影
```

### 场景三：学习艺术风格

```
原始提示词：
一位武士手持长剑

参考图：一张浮世绘风格的插画

生成的风格化提示词：
一位武士手持长剑，浮世绘风格，平面化处理，
传统日本色彩，装饰性线条，艺术插画

提取的变量：
- 风格：浮世绘
- 处理：平面化
- 色彩：传统日本
- 线条：装饰性
```

## 与复刻的区别

| 特性 | 风格学习 | 复刻 |
|------|----------|------|
| 当前提示词 | 保留主体 | 忽略 |
| 分析重点 | 仅风格、构图、色彩 | 整体风格和内容 |
| 适用场景 | 在现有提示词基础上学习 | 从零开始 |

## 常见问题

### 为什么提示必须先输入内容

风格学习的核心是"保留主体，学习风格"。如果没有主体内容，就无法判断哪些是需要保留的，哪些是需要学习的。

### 学习的风格不明显怎么办

- 检查参考图的风格特征是否明显
- 尝试调整提取的变量值
- 使用更风格化的参考图

### 可以同时学习多个参考图的风格吗

当前版本一次只能上传一张参考图。如果需要融合多种风格，可以：

1. 先学习第一个参考图的风格
2. 应用到提示词
3. 再上传第二个参考图继续学习

### 学习后如何进一步优化

- 应用到当前提示词后，可以继续使用优化功能
- 调整变量值来微调风格强度
- 结合测试功能验证效果

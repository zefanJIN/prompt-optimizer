# 文生图复刻

文生图复刻功能可以从参考图反推提示词和变量，适合从喜欢的图像生成类似风格的作品。

## 什么时候用

- 看到喜欢的参考图，想生成类似风格的图像
- 想学习某个图像的风格、构图或色彩
- 需要从参考图提取可复现的提示词

## 怎么用

### 步骤

1. 进入文生图工作区
2. 上传参考图（点击"参考图"按钮）
3. 选择 **复刻** 模式
4. 点击开始处理
5. AI 分析参考图并生成提示词
6. 预览生成结果
7. 点击 **应用到当前提示词**

### 界面说明

- **参考图区域**：显示上传的参考图缩略图
- **生成后的提示词**：AI 反推出的提示词文本
- **提取到的变量**：从图像中识别出的可变参数
- **应用按钮**：将结果写入当前工作区

## 工作原理

### 分析过程

AI 会分析参考图的：

- 主体内容和构图
- 色彩搭配和色调
- 光影效果和氛围
- 风格特征和艺术手法

### 产出物

分析完成后会生成：

- **提示词**：描述图像内容和风格的完整提示词
- **变量**：从图像中提取的可变参数（如主体、风格、色调等）

### 变量处理

- 提取的变量会自动填入默认值
- 变量名具有语义（如"主体"、"风格"、"色调"）
- 可以修改变量值来调整生成效果

## 使用场景

### 场景一：复刻艺术风格

```
参考图：一张赛博朋克风格的城市夜景

生成的提示词：
一座未来城市的夜景，霓虹灯闪烁，高楼林立，雨后的街道反射着彩色灯光，
赛博朋克风格，高对比度，紫色和蓝色为主色调

提取的变量：
- 主体：未来城市
- 风格：赛博朋克
- 色调：紫色和蓝色
- 氛围：雨夜
```

### 场景二：学习构图方式

```
参考图：一张人物肖像，采用黄金分割构图

生成的提示词：
一位年轻女性的肖像，采用黄金分割构图，柔和的自然光，
浅景深，温暖的色调，摄影风格

提取的变量：
- 主体：年轻女性
- 构图：黄金分割
- 光线：柔和自然光
- 风格：摄影
```

## 与风格学习的区别

| 特性 | 复刻 | 风格学习 |
|------|------|----------|
| 当前提示词 | 忽略 | 保留主体 |
| 分析重点 | 整体风格和内容 | 仅风格、构图、色彩 |
| 适用场景 | 从零开始 | 在现有提示词基础上学习 |

## 常见问题

### 生成的提示词不准确怎么办

- 在预览对话框中编辑生成结果
- 调整提取的变量值
- 重新生成（可能会得到不同结果）

### 可以复刻任何图像吗

- 支持 PNG/JPEG 格式
- 文件大小不超过 10MB
- 复刻效果取决于图像的清晰度和特征明显程度

### 复刻后如何进一步优化

- 应用到当前提示词后，可以继续使用优化功能
- 调整变量值来微调生成效果
- 结合测试功能验证效果

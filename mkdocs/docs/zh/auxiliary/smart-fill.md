# 变量智能填充

变量智能填充使用 AI 自动为空变量生成合理值，适合快速测试和验证变量化提示词。

## 什么时候用

- 提示词有多个变量，只想填写核心变量
- 想快速测试提示词效果，不想手动填写所有变量
- 不确定某个变量应该填什么值

## 怎么用

### 批量填充

1. 在变量面板填写核心变量（如"产品名称"）
2. 点击变量面板顶部的 **智能填充变量值** 按钮
3. AI 自动为所有空变量生成值
4. 预览生成结果
5. 确认应用

### 单变量填充

1. 在变量面板找到目标变量
2. 点击变量行的下拉菜单
3. 选择 **智能填充** 选项
4. AI 只为这一个变量生成值
5. 预览并确认

## 工作原理

### 变量识别

- **只填充空变量**：已填写的变量不会被覆盖
- **已填写变量作为上下文**：AI 会参考已填写变量来生成协调的值

### AI 推导逻辑

AI 会根据以下信息推导变量值：

- 提示词的主题和风格
- 变量在提示词中的语义角色
- 已填写变量构成的场景
- 生成值之间的相互协调性

### 预览对话框

生成后会弹出预览对话框，显示：

- 每个变量的生成值
- 生成理由
- 置信度
- 可编辑生成结果
- 可选择要应用的变量

## 前置条件

- 需要配置评估模型（在模型管理中设置）
- 提示词不能为空

## 使用场景

### 快速测试

```
提示词：写一篇关于{{主题}}的文章，面向{{受众}}，风格为{{风格}}

手动填写：主题 = 人工智能
智能填充：受众 = 科技爱好者，风格 = 专业但易懂
```

### 场景构建

```
提示词：为{{产品}}撰写发布文案，目标用户是{{用户群体}}，强调{{核心卖点}}

手动填写：产品 = Prompt Optimizer
智能填充：用户群体 = AI 开发者和提示词工程师，核心卖点 = 一键优化提示词
```

## 常见问题

### 为什么没有生成值

- 检查是否配置了评估模型
- 确认提示词不为空
- 检查是否所有变量都已填写（批量模式下）

### 如何提高生成质量

- 填写核心变量时尽量具体
- 变量名要有语义（如"产品名称"而非"变量1"）
- 提示词本身要清晰描述变量的用途

### 生成的值不合适怎么办

- 在预览对话框中编辑生成结果
- 取消勾选不想应用的变量
- 重新生成（可能会得到不同结果）

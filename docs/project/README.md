## 🚀 核心特性

### 提示词优化
- 智能优化：一键优化提示词，支持多轮迭代
- 双模式优化：系统提示词和用户提示词
- 分析与对比评估：单结果评估和多结果对比

### 图像生成
- 文生图（T2I）：通过文本生成图像
- 图生图（I2I）：基于本地图片变换优化
- 多图生图：多图输入约束生成目标
- 风格迁移：从参考图学习风格

### Prompt Garden
- 提示词发现：浏览社区优质提示词
- 导入与收藏：带元数据和来源绑定
- 版本历史：追踪提示词演变

### 智能收藏
- 资源感知资产：版本历史、可复现示例
- 媒体支持：图片和封面
- 完整备份：连同引用资源一起迁移
